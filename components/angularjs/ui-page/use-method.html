当前位置：常用组件 / AngularJS指令 / ui-page碎片页切换指令
使用方法
使用方法很简单，下面先给出html写法： 
<div ui-pages='trade'>

    <div ui-page href='html/trade/input.html'></div>
    <div ui-page href='html/trade/output.html'></div>

</div>
初始化的时候默认打开第一页，如果想跳页，比如这里进入output.html，只需要引入$targets服务，并执行下列语句：
$targets("trade","#2")';
第一个参数是和你要控制哪一组对应（也就是传递你需要控制的那组的ui-pages属性值），第二个参数是代表第几页。
注意点
1.使用之前，注意修改定义指令的模块名称为项目中定义的名称；
2.一个v-pages和孩子结点中含有v-page的为一组，控制跳页是一组中的动作（就是可以切换的是一组中此刻谁来显示）。
