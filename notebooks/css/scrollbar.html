当前位置：文档笔记 / 不同浏览器常用的滚动条样式修改
webkit系列浏览器
::-webkit-scrollbar
滚动条整体部分
::-webkit-scrollbar-track
滚动条的轨道（里面装有Thumb）
::-webkit-scrollbar-thumb
滚动条里面的小方块，能向上向下移动（或往左往右移动，取决于是垂直滚动条还是水平滚动条）
::-webkit-scrollbar-button
滚动条的轨道的两端按钮，允许通过点击微调小方块的位置
::-webkit-scrollbar-corner
边角，即两个滚动条的交汇处
::-webkit-resizer
两个滚动条的交汇处上用于通过拖动调整元素大小的小控件，比如你使用textarea标签的时候出现的那个用来拖拽大小的东西
:horizontal
用来标记是水平方向上的，比如::-webkit-scrollbar-track:horizontal，选中了水平滚动条的轨道
:vertical
用来标记是垂直方向上的
:decrement
用来标记递减的按钮或轨道碎片，比如::-webkit-scrollbar-button:decrement，选中的是点击递减的按钮
:increment
用来标记递增的按钮或轨道碎片
:start
用来标记位于滑块左边或上边的按钮或轨道碎片，比如::-webkit-scrollbar-button:decrement:start，选中的是点击递减且位于滑块左边和上边的按钮
:end
用来标记位于滑块右边或下边的按钮或轨道碎片
IE系列浏览器
IE浏览器对滚动条修改的能力有限，只可以改改颜色，下面给出一个例子： 
* {

    scrollbar-arrow-color: #f00;/*三角箭头的颜色*/

    scrollbar-face-color: #0f0;/*立体滚动条的颜色*/

    scrollbar-3dlight-color: #00f;/*立体滚动条亮边的颜色*/

    scrollbar-highlight-colofr: #ff0;/*滚动条空白部分的颜色*/

    scrollbar-shadow-color: #0ff;/*立体滚动条阴影的颜色*/

    scrollbar-darkshadow-color: #f0f;/*立体滚动条强阴影的颜色*/

    scrollbar-track-color: #fff;/*立体滚动条背景颜色*/

    scrollbar-base-color: #000;/*滚动条的基本颜色*/

}
