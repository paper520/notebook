搭建项目
Angular cli是一个命令行界面工具，它可以创建项目、添加文件以及执行一大堆开发任务，比如测试、打包和发布，这里的项目搭建就是基于这个命令。
开始项目前，你需要先安装node和npm，然后执行npm install -g @angular/cli安装Angular CLI。
先执行下面的命令初始化项目（其中--skip-install表示node包先不安装，接着使用cnpm install安装会快多了）：
ng new newApp --skip-install
接着进入项目文件夹，安装node包：
cnpm install
启动项目（浏览器会自动打开页面）：
ng serve --open
此时，项目就搭建成功了。
