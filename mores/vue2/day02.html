实例对象
vue的应用都是从创建vue实例对象开始的，实例对象负责管理选择的区域，更多的技术都是在这个区域里面被这个实例对象使用，因此，先来看看这个实例对象。
创建实例对象的方法就是new一个vue对象，基本的结构如下： 
var vm=new Vue({

    el: '挂载点',

    render:function(createElement) { 
        //可选，返回创建好的模板
    },

    template:"可选，和render不可以同时存在，参数是html标签",

    beforeCreate:'可选，生命周期钩子，是一个函数，本文结尾图中别的钩子类似',

    data() {//可选
        return {
            //键值对，双向绑定的数据
        }
    },

    ......

});
实例对象创建以后，在其管理的区域中根据提供的API就可以实现需要的效果了；需要明白的是所有的Vue组件都是Vue实例，并且接受相同的选项对象(一些根实例特有的选项除外)。
