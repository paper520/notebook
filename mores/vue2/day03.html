计算属性
现在有一个数据，如下： 
data() {
    return {

        message: 'yelloxing'

    }
}
如果你使用其在页面上及时显示数据（单向绑定），你可以这样：
{{message}}
如果想及时显示message经过一定改造后的数据，你可以这样：
{{ message.split('').reverse().join('') }}
不过这样会导致模板本来只是用来显示数据，现在充满了过于复杂的逻辑，这时候你可以选择计算属性。
计算属性用的是vue实例对象的computed选项如下配置： 
computed: {
    
    //计算属性的getter
    reversedMessage: function () {
    
        //'this'指向vm实例
        return this.message.split('').reverse().join('');
    
    }

}
此时，使用如下reversedMessage显示就可以实现同样的效果：
{{reversedMessage}}
计算属性默认使用函数代表的是getter，你也可以明确使用getter和setter，例子如下： 
computed: {
                    
    reversedMessage:{
                    
        set:function(newValue){
            //todo
        },
        get:function(){
            //todo
        }
                    
    }
                
}
侦听器
Vue提供了一种更通用的方式来观察和响应Vue实例上的数据变动：侦听属性。对比上面的例子，你可以监听message的改变，然后实时修改，代码如下： 
watch: {

    message:function(val){
        this.reversedMessage=val.split('').reverse().join('');
    }

}
