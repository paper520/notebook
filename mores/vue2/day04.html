基本说明
组件是可复用的Vue实例对象，且带有一个名字，自定义一个组件的方法有很多，一个共同点是：嵌套组合形成一个完整的页面。
自定义组件
除了new Vue()的方法外，还可以通过.vue文件、new Vue()中的选项components和Vue.component()等来定义组件。
同时，因为组件名称意义的模糊，没有固定的形式，可以用来形成页面模板，有效控制效果的等类似产物都可以称为组件，下面来说说几种定义方法。
单文件组件
也就是通过.vue文件创建组件，比如现在有一个组件message.vue，代码如下（更具体的细节此处不涉及，下同）： 
//模板
<template>
    <p>{{message}}</p>
</template>

//控制
<script>
export default {
    data: function() {
        return {
            message: "欢迎来到vue2的世界！"
        };
    }
};
</script>

//样式
<style scoped>
p {
    color: red;
}
</style>
这样一个组件就定义好了，使用前，先要导入进来：
import Message from './message.vue';
这样，Message就是可以使用的组件了，比如在路由在使用，或者作为createElement()函数的参数： 
var vm = new Vue({

    ......
    
    render: function (createElement) {

        return createElement(Message);

    }
                    
});
component方法之全局注册
Vue对象提供了component方法，可以直接定义一个组件，例子如下（代码在./study.component.js中）： 
import Vue from 'vue';
                
Vue.component('study-component', {

    template: "<em>{{messages}}</em>",

    data() {
        return {
            messages: 'vue学习小基地'
        };
    }

});
同样的，在使用前，需要引入：
import './study.component.js';
因为组件是全局注册的，你可以在任何地方通过<study-component></study-component>标签使用这个组件。
component方法之局部注册
同样刚刚的例子，比如在一个组件中需要使用一个局部定义的组件，用例如下： 
import Vue from 'vue';
                
var study ={
    template: "<em>{{messages}}</em>",
    data() {
        return {
            messages: 'vue学习小基地'
        };
    }
};
                
Vue.component('show-component', {

    template: "<div><study-component></study-component></div>",

    components:{
        'study-component':study
    }

});
需要注意的是局部注册的组件在其子组件中不可用
