基本概念和说明
严格的说，vue就是一个JavaScript库，你把他看成一个框架也无伤大雅，反正就是一个提供了很多便捷的前端开发的东西，类似Jquery，不过提供的东西更丰富了。
当你使用他提供的接口或者说规范，反正就是在他规定的世界观里面开发，他会帮你实现你想要的功能，而他的世界里面，仿佛一切都没有原生那么痛苦。
因此，归结到本质还是浏览器本身语言系列（HTML+CSS+JavaScript）和一些开发技巧，再加上vue的世界观就是vue了。
搭建开发环境流程
因为vue2使用了一些目前浏览器还不完全支持的ES6语法，当然也包括.vue等特殊文件，因此我们需要把写的代码使用webpack进行编译，之后浏览器才可以正常运行。
搭建的大致流程是：
  • 1.安装vue2和编译程序需要的node包；
  • 2.配置webpack编译过程；
  • 3.创建一个简单的用例来测试搭建是否成功；
  • 4.运行。
安装node包
我们需要安装的node包分为二部分，我习惯性的把他们分别放在devDependencies和dependencies里面【package.json】。 
"devDependencies": {

    "babel-core": "^6.26.0",

    "babel-loader": "^6.4.1",

    "css-loader": "^0.28.11",

    "vue-loader": "^11.3.4",

    "vue-template-compiler": "^2.5.16",

    "webpack": "^2.4.1",

    "webpack-dev-server": "^2.4.2"

},
"dependencies": {

    "vue": "^2.2.6"

}
既然是vue开发，因此vue包是必须的，webpack是打包框架，webpack-dev-server是服务器，vue-loader和babel-loader分别是用来解析.js和.vue文件的，余下的都是这二个包的依赖文件。
配置webpack
webpack需要配置打包的入口文件和出口文件，然后针对特殊文件需要配置如何去解析【webpack.config.js】。 
var webpack = require('webpack');

module.exports = {
    entry: ['./src/entry.js'],
    output: {
        path: __dirname,
        filename: 'build/main.js'
    },
    resolve: {
        alias: {
            // vue.js = vue.common.js + compiler.js，而如果要使用 template 这个属性的话就一定要用 compiler.js
            'vue': 'vue/dist/vue.js'
        }
    },
    module: {
        loaders: [{
                test: /\.vue$/,
                loader: 'vue-loader'
            },{
                test: /\.js$/,
                loader: 'babel-loader',
                exclude: /node_modules/
            }
        ]
    }
};
在这里，入口文件是entry.js，打包的结果存放在main.js，然后配置了如何解析.vue和.js文件
添加用例
到目前为止，开发环境就搭建完成了，接着来添加一个简单的用例：在页面上通过vue2显示「欢迎来到vue的世界！」。
vue2是通过vue对象来管理一块区域的显示和交互的，具体管理哪一块区域是通过页面上的挂载点来标记。在项目根目录里新建一个index.html，并添加下面二行代码： 
<div id="root">
    {{infomation}}
</div>

<script src="./build/main.js"></script>
因为最终打包的文件放在main.js，因此需要引入，而标签id=root就是定义的挂载点。
接着，还需要一个vue对象，来看看entry.js代码： 
import Vue from 'vue';
                
//vue对象
var vm = new Vue({

    //匹配挂载点
    el: '#root',

    //数据
    data:{
        'infomation':'欢迎来到vue的世界！'
    }

});
el: '#root'标明了该vue对象管理的区域，data里面的就是数据，index.html里面的{{infomation}}双向绑定显示出来。
启动运行
在webpack.config.js中添加如下代码： 
"scripts": {

    "start": "node_modules/.bin/webpack-dev-server --inline --hot --port 20000",

    "release": "node_modules/.bin/webpack"

}
你可以通过npm run release启动编译，然后在浏览器中打开index.html查看效果；或者运行npm start，直接通过浏览器访问127.0.0.1:2000查看效果（前者是编译，后者是热编译，就是修改后立刻生效）。
