当前位置：更多 / 框架或类库 / vue2系统学习随手笔记 / [07] 通过$emit向父级组件发送消息
基本使用方法
子组件如果想传递数据给父组件是通过$emit来实现的，先来看一下子组件代码： 
import Vue from 'vue';
            
Vue.component('emit-comp',{
    
    template:'<button v-on:click="$emit(\'do-self\',\'传递给父组件的值\')">点击传递数据给父组件</button>'

});
接着，看一下使用这个组件的代码： 
......            
    <emit-comp v-on:do-self="doIt($event)"></emit-comp>
......
    methods: {
        doIt: function(param) {
            alert("显示：" + param);
        }
    }
......
子组件通过$emit触发父组件一个自定义事件，父组件提前注册好处理方法，父组件可以通过$event获取子组件传递的参数。
如果在父组件中把组件使用代码改成如下：
<emit-comp v-on:do-self="doIt"></emit-comp>
那么，doIt函数的第一个参数就是传递过来的数据。
