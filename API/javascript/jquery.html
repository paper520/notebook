语法 用例 例子描述
append() $('body').append('<a>小笔记<a/>'); 在body元素的内部结尾插入"<a>小笔记<a/>"
prepend() $('body').prepend('<a>小笔记<a/>'); 在body元素的内部开头插入"<a>小笔记<a/>"
after() $('div').after('<a>小笔记<a/>'); 在div元素后面插入"<a>小笔记<a/>"
before() $('div').before('<a>小笔记<a/>'); 在div元素前面插入"<a>小笔记<a/>"
remove() $("p").remove(".target"); 删除class包含target的div元素（及其子元素），参数是过滤作用，可选
empty() $('div').empty(); 删除div元素内部的全部内容
width() $('div').width().width('100px'); 先获取div宽，再设置div宽为100px(不包括内边距、边框或外边距）
height() $('div').height().height('100px'); 先获取div高，再设置div高为100px(不包括内边距、边框或外边距）
innerWidth() $('p').innerWidth(); 获取p标签的宽（包括内边距）
innerHeight() $('p').innerHeight(); 获取p标签的高（包括内边距）
outerWidth() $('p').outerWidth(); 获取p标签的宽（包括内边距和边框）
outerHeight() $('p').outerHeight(); 获取p标签的高（包括内边距和边框）
addClass() $('em').addClass('warn'); 给em标签添加一个名称是warn的class
removeClass() $('em').removeClass('warn'); 删除em标签一个名称是warn的class
toggleClass() $('em').toggleClass('warn'); 如果em标签有名称为warn的class就删除，没有添加
hasClass() $('em').hasClass('warn'); 如果em标签有名称为warn的class就返回true，没有返回false
