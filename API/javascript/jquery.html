语法 用例 例子描述
append() $('body').append('<a>小笔记<a/>'); 在body元素的内部结尾插入"<a>小笔记<a/>"
prepend() $('body').prepend('<a>小笔记<a/>'); 在body元素的内部开头插入"<a>小笔记<a/>"
after() $('div').after('<a>小笔记<a/>'); 在div元素后面插入"<a>小笔记<a/>"
before() $('div').before('<a>小笔记<a/>'); 在div元素前面插入"<a>小笔记<a/>"
remove() $("p").remove(".target"); 删除class包含target的div元素（及其子元素），参数是过滤作用，可选
empty() $('div').empty(); 删除div元素内部的全部内容
replaceWith() $('div').replaceWith("<a>小笔记<a/>"); 把div标签[包括内部内容]替换成<a>小笔记<a/>
语法 用例 例子描述
width() $('div').width().width('100px'); 先获取div宽，再设置div宽为100px(不包括内边距、边框或外边距）
height() $('div').height().height('100px'); 先获取div高，再设置div高为100px(不包括内边距、边框或外边距）
innerWidth() $('p').innerWidth(); 获取p标签的宽（包括内边距）
innerHeight() $('p').innerHeight(); 获取p标签的高（包括内边距）
outerWidth() $('p').outerWidth(); 获取p标签的宽（包括内边距和边框）
outerHeight() $('p').outerHeight(); 获取p标签的高（包括内边距和边框）
语法 用例 例子描述
addClass() $('em').addClass('warn'); 给em标签添加一个名称是warn的class
removeClass() $('em').removeClass('warn'); 删除em标签一个名称是warn的class
toggleClass() $('em').toggleClass('warn'); 如果em标签有名称为warn的class就删除，没有添加
hasClass() $('em').hasClass('warn'); 如果em标签有名称为warn的class就返回true，没有返回false
语法 用例 例子描述
parent() $("span").parent(); 查找span标签的父元素，可以带参数过滤
parents() $("span").parents(); 查找span标签的祖先元素，可以带参数过滤
parentsUntil() $("span").parentsUntil("div"); 返回span开始到div为止的祖先元素
children() $("i").children(); 返回i元素的所有直接子元素，可以带参数过滤
find() $("i").find(); 返回i元素的所有后代元素，可以带参数过滤
siblings() $('nav').siblings(); 返回nav元素的所有同胞元素，可以带参数过滤
next() $('nav').next(); 返回nav元素的下一个同胞元素，可以带参数过滤
nextAll() $('nav').nextAll(); 返回nav元素的之后的全部同胞元素，可以带参数过滤
nextUntil() $("h2").nextUntil("h6"); 返回介于 <h2>与<h6>元素之间的所有同胞元素[方向向下]
prev() $('nav').prev(); 返回nav元素的前一个同胞元素，可以带参数过滤
prevAll() $('nav').prevAll(); 返回nav元素的之前的全部同胞元素，可以带参数过滤
prevUntil() $("h2").prevUntil("h6"); 返回介于 <h2>与<h6>元素之间的所有同胞元素[方向向上]
语法 用例 例子描述
first() $('div').first(); 返回选中的div元素中的第一个
last() $('div').last(); 返回选中的div元素中的最后一个
eq() $('div').eq(3); 返回选中的div元素中的第四个
filter() $('p').filter(".target"); 返回带有类名"target"的所有p元素
not() $("p").not(".target"); 返回不带有类名"target"的所有p元素
语法 用例 例子描述
text() $('div').text(); 返回div元素里面的文本内容，带值是设置
html() $('div').html(); 返回div元素里面的html内容，带值是设置
val() $('input').val(); 返回input的值，带值是设置
attr() $('input').attr('name'); 返回input的name属性的值，有第二个参数是设置
上面方法都拥有回调函数，回调函数有两个参数：被选元素下标，以及原始值，你可以以函数新值返回您希望使用的字符串[例如：$("#test1").text(function(i,origText){return '新值';});]
语法 用例 例子描述
css("key"); $('header').css('color'); 返回header样式的color
css("key","value"); $('header').css('color','blue'); 设置header标签样式的color为blue
css({"key":"value","key":"value",...}); $('header').css({'color':'blue','fontSize':'14px'}); 设置header的color和font-size
语法 用例 例子描述
bind(); $("button").bind("click",function(){}); 给按钮绑定一个点击事件
on(); $('div').on('click','button',function(){}); 给div元素里面的button绑定一个点击事件，此刻里面的button可以不存在
trigger(); $("button").trigger("click"); 触发按钮的点击事件
unbind(); $("button").unbind("click"); 取消按钮绑定的点击事件
