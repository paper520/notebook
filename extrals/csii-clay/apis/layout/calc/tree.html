tree 画树辅助计算

绘制树图的时候，布局目前只提供了一种基础模型计算后的位置数据，为了在实际开发的时候更快速开发，提供了绘制一些常见树的辅助计算方法。

获取计算对象 

var treeCalc=$$.treeCalc(config);

config 配置

config用于配置一些参数，告诉获取的对象，如何计算结点位置等，其是一个json，下面是配置的选项，全部必输：

1.图形类型：左到右、右到左、上到下、下到上或圆 

"type": "LR|RL|TB|BT|circle"

2.如果图形类型是LR|RL|TB|BT，需要设置下面二个参数 

// 分别表示图形的宽和高
"width":number
"height":number

3.如果图形类型是Circle，需要配置下面三个参数 

// 表示图形半径
"radius":number
// 圆心位置
"cx":number
"cy":number

使用计算对象

配置好了以后，就可以多次重复使用了： 

var nodes=treeCalc(nodes, size);

返回的数据就是计算后的结点位置。

nodes, size分别来自treeLayout布局的drawer回调参数，请具体查看tree布局一节。
