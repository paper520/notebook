arc 画圆弧辅助计算

绘图pie图或者单纯的绘制一个圆弧的时候，有时需要在圆弧上添加提示文字，或带连线的提示文字，文字或连线的位置等计算，就是这里需要解决的问题。

必须配置 

/**
 * 准备arc计算对象
 * */
var arcCalc = $$.arcCalc()

// 设置内外半径
.setRadius(r1, r2)

// 设置弧中心
.setCenter(cx, cy)

// 设置圆弧起点弧度，和旋转弧度
.setDeg(beginA, rotateA);

计算接口

"质心" 

var centroid = arcCalc.getCentroid();

文字连线 

var lines = arcCalc.getLines(line1_len, line2_len);

文字连线的意思是，计算二条折线的三个点的位置，需要二个参数，分别表示二段折线的长度，返回三个点的位置。
