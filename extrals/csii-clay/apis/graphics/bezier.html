贝塞尔曲线 bezier

这里把贝塞尔曲线限制在只有二个控制点，主要是用于绘制树图时候的连线，因此分为二种类型（比如直方树和圆树）。

Canvas2D 

var bezier=$$.canvas.bezier('canvas')
    // 设置曲线类型
    .setType(ax, ay, type)
    // 设置控制把柄的长度
    .setL(length);

其中，设置曲线类型方法，可选类型有二种（缺省类型是"normal"）：

  • 1.type="normal",(ax,ay)是参考方向
  • 2.type="circle",(ax,ay)是参考中心
bezier(bx, by, ex, ey).stroke();

这里再强调一下，最后一定别忘了调用stroke方法，后面不再强调了。

SVG 

var bezier=$$.svg.bezier()
    // 设置曲线类型
    .setType(ax, ay, type)
    // 设置控制把柄的长度
    .setL(length);

参数设置和上面的一样，最后绘制图形： 

$$('path').attr('d', bezier(bx, by, ex, ey))
    .css({
        'key':'value',
        ...
});
